# 锻造师 - 游戏说明

> 版本: v1.0.12

---

## 游戏简介

**锻造师**是一款融合创意绘画与动作竞技的 2D 休闲游戏。你将亲手绘制武器、选择材质、经历完整的锻造流程，最终在试炼场中用自己打造的武器击败锻造师。

---

## 游戏流程

游戏分为六个环节，按阶段递进：

```
阶段五: 绘制武器 → 阶段四: 选择材质 → 阶段三: 锤击锻打
→ 阶段二: 淬火冷却 → 阶段一: 砥砺打磨 → 试炼: 试炼场
```

每个环节开始前会显示操作说明，点击"开始"按钮进入。

---

## 环节详解

### 1. 绘制武器（阶段五）

在画布上自由绘制你想要的武器形状。系统会根据形状特征自动识别武器类型：

| 武器类型 | 识别特征 | 战斗风格 |
|----------|----------|----------|
| 剑 | 细长形状，有尖端 | 均衡型：攻速与伤害平衡 |
| 斧 | 宽窄变化大，有厚重部分 | 爆发型：慢速高伤害 |
| 矛 | 极细长（宽高比>4） | 风筝型：极快极远 |
| 盾 | 近圆形/方形闭合图案 | 控制型：低伤高击退 |
| 钩 | 多个尖角转折 | 连击型：拉近+全方位扫击 |
| 奇物 | 无法归类的形状 | 特殊型：单招但材质加成高 |

**操作提示**：
- 尽量画得清晰完整
- 可使用底部的预设模板快速选择
- 画错可点击"撤销"或"清空"

**隐藏机制**：画出多笔画且相互连接的复合形状，可解锁"复合武器"，拥有两种武器形态可切换！

### 2. 选择材质（阶段四）

彩色方块从屏幕右侧高速飞过，点击你想要的材质方块即可选中。

| 材质 | 效果 | 适合搭配 |
|------|------|----------|
| 黑铁 | 攻击+20%，击退+50% | 重型武器（斧/盾） |
| 秘银 | 攻速+25%，攻击时不减速 | 灵活武器（剑/矛） |
| 黑曜石 | 攻击+40%，但受伤+20% | 高风险高回报玩法 |
| 血矿 | 伤害15%转化为回血 | 持久战玩法 |
| 陨星 | 每秒2%真实伤害（灼烧） | 风筝/消耗流 |
| 逆鳞 | 格挡时反弹100伤害 | 防反流（盾/钩） |
| 灵木 | 连击每段+5%伤害（上限50%） | 连击型武器（钩/剑） |

**提示**：5秒内未选择将自动随机分配。

### 3. 锤击锻打（阶段三）

节奏小游戏。屏幕上有一个来回移动的光标和黄色判定区域：
- **Perfect**（精准命中中心）：100 分
- **Good**（命中黄色区域）：70 分
- **Miss**（未命中）：20 分

共需要锤击 5 次，在光标经过黄色区域时点击屏幕。

### 4. 淬火冷却（阶段二）

温度控制小游戏：
- **按住屏幕**：武器持续降温（-200度/秒）
- **松开**：停止降温
- **目标**：将温度降至目标区间内松手

注意不要过冷！精确控温获得高分。

### 5. 砥砺打磨（阶段一）

在方块区域内来回滑动手指/鼠标：
- 完成一个来回 = 一次打磨
- 3秒内尽可能多地完成打磨
- 6次及以上即为满分

### 6. 试炼场（试炼）

横版动作战斗！60秒内击败锻造师获得胜利。

## 试炼场战斗技巧

### 基础技巧

- **连击奖励**：连续命中敌人不间断，连击数越高得分越多
- **利用平台**：场景中有多个平台，可利用高低差躲避和进攻
- **攻击减速**：攻击时移动速度减半（秘银材质除外），注意走位

### 武器特性利用

| 武器 | 招式1 | 招式2 | 战术建议 |
|------|-------|-------|----------|
| 剑 | 横斩（快） | 上挑（中） | 交替使用，保持连击 |
| 斧 | 劈砍（慢高伤） | 横扫（超大范围） | 等敌人靠近再出手 |
| 矛 | 突刺（极快极远） | 横扫（大范围） | 保持距离持续输出 |
| 盾 | 盾击（超强击退） | 冲撞（前冲） | 控制距离，安全输出 |
| 钩 | 钩击（拉近敌人） | 回旋（360度） | 先钩后旋，高伤连招 |

### 对阵锻造师

- 锻造师会主动追击你并发起攻击
- 锻造师使用剑类攻击，注意躲避
- 武器碰撞会触发格挡（你被弹开，攻击中断）
- 利用平台制造距离，找准空档进攻

## 品质系统

锻造评分（三个环节平均分）决定武器品质：

| 品质 | 评分要求 | 攻击倍率 |
|------|----------|----------|
| 粗制 | 0~29 | 1.0x~1.29x |
| 普通 | 30~49 | 1.30x~1.49x |
| 精良 | 50~74 | 1.50x~1.74x |
| 史诗 | 75~89 | 1.75x~1.89x |
| 传说 | 90~100 | 1.90x~2.0x |

品质越高，武器攻击力越强！认真完成每个锻造环节很重要。

## 排行榜

击败锻造师后可输入昵称提交成绩至全服排行榜。排名规则：
1. 用时越短排名越高
2. 用时相同时，总伤害越低排名越高（效率优先）

## 设置选项

在主菜单右上角齿轮按钮可进入设置面板：

- **按键绑定**：自定义所有 PC 按键
- **环节时长**：调整各环节的时间限制
  - 试炼时长：30/45/60/90/120 秒
  - 选材时长：3/5/8/10/15 秒
  - 锤击时长：3/5/8/10/15 秒
  - 淬火时长：3/5/8/10/15 秒
  - 砥砺时长：2/3/5/8/10 秒

## 常见问题

**Q: 为什么画出的武器不是我想要的类型？**
A: 系统根据形状特征自动识别。细长有尖端=剑，近圆形闭合=盾，宽窄变化大=斧。可参考预设模板了解各类型特征。

**Q: 复合武器怎么触发？**
A: 画出多笔画（至少2笔），其中至少1个闭合图形，且笔画之间端点距离小于30px（视觉上看起来连接在一起）。

**Q: 试炼场怎么格挡？**
A: 格挡是被动触发的——当你的武器和锻造师的武器在攻击时碰撞，就会产生格挡效果。你会被弹开，攻击中断。所以要注意攻击时机，避开敌人的攻击。

**Q: 材质方块飞太快抓不到？**
A: 方块是高速弹幕，需要快速反应。可以提前观察方块颜色，在它经过你附近时点击。也可以在设置中增加选材时长。

**Q: 游戏支持横屏还是竖屏？**
A: 仅支持横屏模式。竖屏时会显示旋转提示。
